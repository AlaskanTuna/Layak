# Layak Demo Video Scripts

This folder keeps the finalized demo-video recording assets together in one place.

## Script Options

- [3-Minute Director Script](./demo-video-director-script-3min.md)
  Judging-safe cut optimized for the stricter `3-minute` FAQ reading. Best when the team wants the most compact version focused on the live v1 agentic flow.
- [5-Minute Director Script](./demo-video-director-script-5min.md)
  Story-led cut optimized for the handbook's `up to 5 minutes` allowance. Best when the team wants more time for Aisyah's problem, the product objective, and the pipeline story before the live walkthrough.

## How To Use These Files

- Read only the `VOICEOVER` sections aloud.
- Follow only the `ON SCREEN` sections while recording.
- Use `PACE` for pauses and emphasis.
- Use `EDITOR NOTE` for fallback takes, crops, or cleaner edits.

## Shared Message Guardrails

- Layak is a `Track 2: Citizens First` submission.
- The core demo story is `v1`: intake, five-step pipeline, ranked results, provenance, and draft packets.
- `v2` is the surrounding product layer: sign-in, saved evaluations, quotas, settings, language sync, export/delete, and history.
- Aisyah's current demo flow surfaces `6 matched items` in total:
  `5 benefit schemes` that contribute to upside, plus `1 required contribution` (`PERKESO SKSPS`) shown separately.
- The outputs are `grounded`, `draft only`, and still `submitted manually` by the user.

## Recommended Recording Order

1. Capture a short results-only take for the cold open.
2. Capture one clean end-to-end app run.
3. Capture close-up backups for the pipeline, provenance, code execution, or packet area if needed.
