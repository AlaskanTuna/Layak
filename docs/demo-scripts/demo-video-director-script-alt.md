# Layak Demo Video Director Script — Pitch Deck Arc (≤4 Minutes)

Intended runtime: `3:30–3:45`
Hard cap used for planning: `4 minutes` (within the 5-minute handbook allowance)

Intended use: a **pitch-deck-faithful** cut. Follows the narrative order of `docs/slides/layak-pitch-deck.html` so the video and the deck tell the same story.
Best for: submission reel, jury playback, social share.
Read aloud: only the `VOICEOVER` sections.

Why this version exists: the previous alt script was a product tour over live-app footage only. This version was requested because our edit bay has a mixed clip pool — live-app takes **plus** exported slide PNGs in `docs/slides/assets/` — and the easiest way to cut is to follow the deck's story. Slide cutaways are treated as first-class shots here.

Primary objective: land `Impact` and `Innovation` via the problem/outcome framing, prove `AI Implementation` in the pipeline section, and keep `Presentation` consistent with the deck the jury will also see.

## Recording / Edit Setup

- Keep the deck PNGs ready as B-roll: `docs/slides/assets/slide-01-cover.png` through `slide-15-thank-you.png` (3840×2160).
- Live-app takes should already be authenticated and start from `/dashboard/evaluation/upload`.
- Use the `Aisyah` sample path for every live-app clip. Do not try live upload.
- For any beat where the live take is shaky, fall back to the matching slide PNG. This is explicitly allowed — the script is structured so a slide cutaway is never a downgrade.
- Capture one extra results-page take for the cold open (the RM 12,308 hero number must be readable).
- Keep mouse movement slow. No settings, pricing, waitlist, or landing-page browsing.
